# TRELLO-CLONE

### Running Locally

Make sure you have Node and npm install.

  1. Clone or Download the repository 
    <pre>git clone https://github.com/uttampatro/trello-clone.git
    $ cd trello-clone</pre>
  2. Install Dependencies

      <pre>npm install</pre>
  3. Start the Application

     <pre>npm start</pre>
